동작 방식

설치 한 번으로 Claude Code가 글로벌 컨텍스트를 자동으로 읽습니다.

01

설치

SKILL.md 파일이 ~/.claude/skills/ 에 설치됩니다. Jira MCP와 실행 권한도 settings.json 에 자동 등록됩니다.

npx chrys-tools install
02

Claude Code 시작

Claude Code는 시작 시 ~/.claude/CLAUDE.md 와 skills 폴더를 자동으로 읽어 모든 skill이 활성화됩니다.

03

자연어로 요청

어느 프로젝트에서든 티켓 번호, 배포 요청, 커밋 명령을 자연어로 말하면 멀티 에이전트 파이프라인이 작동합니다.

04

병렬 서브에이전트 실행

Haiku 서브에이전트 3개가 동시에 코드 탐색 · Jira 조회 · Git 히스토리를 병렬 처리합니다. 요약본만 메인 컨텍스트로 반환해 토큰 낭비를 방지합니다.

05

결과 보고 & 확인

수정 계획 제시 → 사용자 승인 → 코드 수정 → 영향 범위 분석 → Before/After 리뷰 보고서 순으로 투명하게 진행됩니다.
