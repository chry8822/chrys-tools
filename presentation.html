// how it works

동작 원리

Claude Code는 시작 시 ~/.claude/를 자동으로 읽습니다.
이 패키지는 해당 경로에 skill 파일들을 설치해줍니다.

01

설치 명령어 실행

설치할 기능을 선택하고 Jira · 서버 연결 정보를 입력합니다.

npx chrys-tools install
02

SKILL.md 자동 설치

~/.claude/skills/에 각 기능의 SKILL.md가 설치되고 CLAUDE.md에 참조가 자동 등록됩니다.

03

MCP + 권한 자동 구성

Atlassian MCP 서버와 실행 권한이 ~/.claude/settings.json에 자동 등록됩니다.

04

어디서든 바로 사용

프로젝트 레포에 아무것도 추가하지 않아도, Claude Code를 켜는 순간부터 모든 기능이 동작합니다.
