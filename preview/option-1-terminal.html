프로젝트 루트에 자동 저장

설치 한 번, 어디서나 동작

Claude Code는 시작 시 ~/.claude/를 자동으로 읽습니다.
chrys-tools는 이 경로에 Skills를 설치해 어느 프로젝트에서나 동작합니다.

01

install 실행

설치할 기능 선택 · Jira API Token · 서버 SSH 정보 입력

$ npx chrys-tools install
02

~/.claude/ 자동 구성

CLAUDE.md · settings.json · config.json · skills/ 자동 생성

~/.claude/skills/issue-analyzer/SKILL.md
03

Claude Code에서 바로 사용

어느 프로젝트에서 Claude Code를 열어도 Skills가 자동으로 활성화

"ABEH-1234 수정해줘" · "qa 배포해줘"

30초면 끝납니다

terminal
$ npx chrys-tools install
01 / 기능 선택

Jira 분석 · 서버 배포 · README 생성 중 원하는 기능만 선택

02 / 정보 입력

Jira API Token, 서버 SSH 정보를 대화형 프롬프트로 입력

03 / 바로 사용

설치 완료 후 어느 프로젝트에서나 Claude Code에서 바로 동작

npx chrys-tools add jira
npx chrys-tools config deploy
npx chrys-tools list
